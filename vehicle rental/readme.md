# Vehicle Rental Management System

A desktop-based Vehicle Rental Management System developed using **Python (Tkinter)** for the frontend and **MySQL** for the backend database.  
This project automates the management of vehicles, rental transactions, and availability tracking for a small-scale vehicle rental business.
  
## Features
- Add new vehicles to the system
- Rent vehicles to customers
- Return rented vehicles
- View available vehicles
- View complete rental history
- Dashboard summary (total, rented, available vehicles)
- Simple and user-friendly GUI

## Technologies Used

- **Frontend:** Python (Tkinter)
- **Backend:** Python
- **Database:** MySQL
- **Connector:** mysql-connector-python
